# 🌐 CodeAlpha Language Translator

## 📌 Project Overview
This project is a web-based Language Translator developed using HTML, CSS, and JavaScript. It allows users to translate text between different languages with a simple and user-friendly interface.

## ✨ Features
- Translate text between multiple languages
- Simple and attractive user interface
- Fast and easy to use
- Responsive design for different devices

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📂 Project Structure
CodeAlpha_language-translator/
│── index.html
│── style.css
│── script.js
│── README.md

## 🚀 How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any web browser.
4. Enter text and select languages for translation.
